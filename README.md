# ☕ Meus Estudos em Java - Curso em Vídeo

Este repositório foi criado para armazenar meus códigos, anotações e exercícios práticos desenvolvidos durante os meus estudos da linguagem **Java** com o curso do professor GUstavo Guanabara disponível no youtube em seu canal **Curso em Vídeo**, a fim de complementar as aulas de Programação Orientada a Objetos da Unicamp.

## 📺 Sobre o Curso

O curso é focado em dar uma base sólida sobre a linguagem Java, abordando desde a sua história, como a linguagem funciona por baixo dos panos, até conceitos mais avançados e Orientação a Objetos.

🔗 **Link para a playlist oficial do curso:** [Curso de Java - Curso em Vídeo](https://www.youtube.com/watch?v=v_ZCtgwbS3o&list=PLHz_AreHm4dkI2ZdjTwZA4mPMxWTfNSpR&index=3)

* **Linguagem:** Java (JDK)
